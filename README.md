# React Native Not Uygulaması

React native'yi öğrenmek ve kendimi geliştirmek amacıyla yazmış olduğum bir not uygulamasıdır. Database olarak AsyncStorage kullandım.

### Özellikleri

- Not ekleme.
- Not düzenleme.
- Not silme.
